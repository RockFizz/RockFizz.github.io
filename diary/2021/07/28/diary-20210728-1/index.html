Node.js是一个基于事件驱动I/O的服务端JavaScript环境，基于Google的V8引擎。
NPM的全称是NodePackageManager,是一个NodeJS包管理和分发工具,已经成为了非官方的发布Node模块(包)的标准。NPM是随同NodeJS一起安装的包管理工具。

Nodejs与V8引擎及原理

在window命令行查询node安装目录 

npm config ls  

node -v

万物皆模块，模块其实都是文件，nodejs目前支持es6模块和commonjs模块。 模块主要分为两大类，一类是nodejs提供的模块，称为核心模块，另一类是用户编写的模块，称为文件模块。

核心模块可以理解为nodejs的内置核心功能模块，不用创建和下载，它在nodejs源代码编译的时候就会被编译成二进制文件，nodejs启动时，这些核心模块就会直接被加载进内存，所以核心模块加载时，相对于文件模块，核心模块引用时不需要进行文件定位和动态编译，速度上有一定的优势，如http，fs，path等常用模块都属于核心模块。
核心模块一般定义在nodejs源代码的lib目录下。

UE4模拟Unity的CameraLayer

在Unity中我们知道可以通过为物体赋予不同的Layer，然后可以设置相机的CullingMask属性达到“不渲染”某一层物体的结果。(物体只是不渲染了，碰撞什么的还会继续存在。)

UE4中的Camera默认并没有提供类似的功能，那么我们能不能实现这样的效果呢？当然可以，每一个Actro自带一个OwnerNoSee的属性（在做联网FPS时非常好用），我们可以配合SetOwner节点和这个属性共同作用，将某一个相机设置成当前物体的Owner并设置OwnerNoSee为True即可实现上面的效果。

新建一个Actor蓝图类，定义一个Actor参数（命名为Camera，可以根据你的需要随便命名，但尽量做到见名知义）在ConstuctionScriopt中设置上述属性。这样设置后，从指定的Camera看物体就看不到当前Actor。

纯js手动实现双向数据绑定

双向数据绑定指的就是，绑定对象属性的改变到用户界面的变化的能力，反之亦然。

1.需要一个UI元素和属性相互绑定的方法
2.需要监视属性和UI元素的变化
3.需要让所有绑定的对象和UI元素都能感知到变化

手动绑定：(有点像观察者模式)
通过在数据对象上定义get和set方法，调用时手动调用get或set数据，改变数据后出发UI层的渲染操作；
以视图驱动数据变化的场景主要应用与input、select、textarea等元素，当UI层变化时，通过监听dom的change，keypress，keyup等事件来出发事件改变数据层的数据。整个过程均通过函数调用完成。

<script>
    //显示元素
    var elems = [document.getElementById('el'), document.getElementById('input')];
    //数据对象
    var data = {
      value: 'hello!'  
    };

    var command = {
      //显示元素的内容变化
      text: function(str){
        this.innerHTML = str;
      },
      //数据对象的值变化
      value: function(str){
        this.setAttribute('value', str);
      }
    };

    var scan = function(){    
      //扫描带指令(自定义)的节点属性
      for(var i = 0, len = elems.length; i < len; i++){
        var elem = elems[i];
        elem.command = [];
        for(var j = 0, len1 = elem.attributes.length; j < len1; j++){
          var attr = elem.attributes[j];
          if(attr.nodeName.indexOf('q-') >= 0){
            //调用属性指令，这里可以使用数据改变检测
            //attr.nodeName.slice(2) 自定义属性节点名字截取从第3个到一直到结束的字符串 (此示例为value或text)
            //输入参数为指定dom对象，执行方法根据自定义节点属性
            //指定dom对象elem,执行command对象的方法text(str),参数str为 data[自定义属性节点的值attr.nodeValue]的值，即data['value']
            //指定dom对象elem,执行command对象的方法value(str),参数str为 data[自定义属性节点的值attr.nodeValue]的值，即data['value']
            command[attr.nodeName.slice(2)].call(elem, data[attr.nodeValue]);
            elem.command.push(attr.nodeName.slice(2));
          }
        }
      }
    }

    //设置数据后扫描
    function mvSet(key, value){
      data[key] = value;
      scan();
    }
    //数据绑定监听
    elems[1].addEventListener('keyup', function(e){
      mvSet('value', e.target.value);
    }, false);

    scan();

    //改变数据更新视图
    setTimeout(function(){
      mvSet('value', 'fuck');
    },1000)
</script>
