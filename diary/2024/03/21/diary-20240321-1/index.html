传递指针变量并避免其指向地址和值发生变化

指向常量的指针

void printValue(const int* ptr) {
    std::cout << "Value: " << *ptr << std::endl;
    // 无法通过指针修改其指向的值
}
int main(int argc, char *argv[])
{
    int value = 5;
    const int* ptr = &value; // 指向常量的指针
    printValue(ptr);
    return 0;
}

传递指针的引用

通过传递指针的引用来允许函数修改指针本身所指向的地址

可以确保指针指向的原始对象(值)不变的情况下，修改指针指向新的对象(值) 

#include <iostream>
void updatePointer(int* &ptr) {
    int newValue = 10;
    ptr = &newValue; // 修改指针指向的地址为 newValue 的地址
}
int main() {
    int value = 5;
    int* ptr = &value;

    std::cout << "Original Value: " << *ptr << std::endl;
    updatePointer(ptr); // 传递指针的引用
    std::cout << "Updated Value: " << *ptr << std::endl;

    return 0;
}

c++11智能指针

C++11引入了智能指针，其中最常用的是std::shared_ptr和std::unique_ptr。这些智能指针可以帮助管理动态分配的内存，并在其生命周期结束时自动释放内存，从而避免内存泄漏。

以下是使用std::shared_ptr的例子 

#include <iostream>
#include <memory>
void sharedPtrExample() {
    std::shared_ptr<int> ptr = std::make_shared<int>(5); // 创建一个指向动态分配的整数的shared_ptr
    std::cout << "Value: " << *ptr << std::endl;
    // 当ptr的引用计数变为0时，会自动释放动态分配的内存
}
int main() {
    sharedPtrExample(); // 在函数内部创建的shared_ptr在函数结束后会自动释放内存
    return 0;
}

以下是std::unique_ptr的例子 

#include <iostream>
#include <memory>

void uniquePtrExample() {
    std::unique_ptr<int> ptr(new int(10)); // 创建一个指向动态分配的整数的unique_ptr
    std::cout << "Value: " << *ptr << std::endl;
    // 在函数结束后，unique_ptr将自动释放动态分配的内存
}

int main() {
    uniquePtrExample(); // 在函数内部创建的unique_ptr在函数结束后会自动释放内存
    return 0;
}

std::shared_ptr和std::unique_ptr的区别

  1. 所有权不同
std::unique_ptr 是独占所有权的智能指针，意味着同一时间只能有一个 std::unique_ptr 指向同一块内存。当 std::unique_ptr 被销毁或重置时，它所管理的对象也会被销毁。
std::shared_ptr 允许多个 std::shared_ptr 实例共享同一块内存。它使用引用计数来跟踪有多少个 std::shared_ptr 实例指向同一内存块，只有当最后一个 std::shared_ptr 被销毁时，才会释放内存。
  1. 性能开销不同 
std::shared_ptr 比 std::unique_ptr 更复杂，因为它需要维护一个引用计数，这可能导致一些额外的性能开销。另外，由于引用计数，std::shared_ptr 的内存占用比 std::unique_ptr 更大。
std::unique_ptr 不需要引用计数，因此更加轻量级和高效。
  1. 使用场景不同 
如果需要多个指针共享同一块内存，可以使用 std::shared_ptr。
如果需要确保只有一个指针拥有所有权，并且在离开作用域时自动释放内存，应该使用 std::unique_ptr。

指向常量的指针

无法通过指针修改其指向的值

void printValue(const int* ptr) {
    std::cout << "Value: " << *ptr << std::endl;
    // 无法通过指针修改其指向的值

{
    int value = 5;

常量引用

void printValue(const int& ref) {
    std::cout << "Value: " << ref << std::endl;
    // 无法通过引用修改其引用的值
}

int main() {
    int value = 5;
    printValue(value);
    return 0;
}

深拷贝

通过深拷贝创建对象的副本，避免原始对象的改变影响到指针指向的内容 

void printValue(int* ptr) {
    std::cout << "Value: " << *ptr << std::endl;
    // 可以通过深拷贝创建对象的副本，避免原始对象的改变影响到指针指向的内容
    int copiedValue = *ptr;
    // 对副本进行操作而不影响原始对象
}

int main() {
    int value = 5;
    int* ptr = &value;
    printValue(ptr);
    return 0;
}

由char*打印指定长度的hex字符串

#include <iostream>
#include <iomanip> // 用于设置输出格式

void printHexFromChar(const char* data, int length) {
    std::cout << "Hexadecimal representation of the data (" << length << " bytes): ";
    for (int i = 0; i < length; ++i) {
        // 将每个字符以十六进制格式打印出来
        std::cout << std::hex << std::setw(2) << std::setfill('0') << static_cast<int>(static_cast<unsigned char>(data[i])) << " ";
    }
    std::cout << std::dec << std::endl; // 恢复默认的十进制输出格式
}

int main() {
    char* data = "hello"; // 要打印的 char* 数据
    int length = 5; // 数据的长度

    printHexFromChar(data, length);

    return 0;
}

C++的short(有符号、无符号)保存在byte[]2字节读写

signed short和unsigned short都是16位（2字节）的数据类型，
signed short可以表示负数和正数（-32768到32767），
unsigned short只能表示非负数（0到65535）
C++处理字节数据时通常会使用char或unsigned char数组来表示字节流

signed short到2字节数组的写入和读取

#include <iostream>
// 将signed short值写入到byte数组中（小端序,低位在前）
void writeShortToByteArray(signed short value, unsigned char* byteArray) {
    byteArray[0] = (unsigned char)(value & 0xFF);        // 获取低8位并存储
    byteArray[1] = (unsigned char)((value >> 8) & 0xFF); // 获取高8位并存储
}
// 从byte数组中读取signed short值（小端序）
signed short readShortFromByteArray(const unsigned char* byteArray) {
    return (signed short)(byteArray[0] | (byteArray[1] << 8));
}
int main() {
    signed short originalValue = -12345; // 示例值
    unsigned char byteArray[2]; // 存储signed short的2字节数组
    // 写入
    writeShortToByteArray(originalValue, byteArray);
    // 打印显示字节数组内容的hex字符串形式
    for (int i = 0; i < 2; ++i) {
        std::cout << std::hex << std::setw(2) << std::uppercase <<std::setfill('0') << (int)(unsigned char)byteArray[i] << " ";
    }
    std::cout << std::endl;
    // 读取
    signed short readValue = readShortFromByteArray(byteArray);
    std::cout << "Read signed short: " << std::dec << readValue << std::endl;
    return 0;
}
