配置环境第一步：安装pcl

在官网下载exe安装包和如图zip文件 

PCL-1.12.1-AllInOne-msvc2019-win64.exe  
pcl-1.12.1-pdb-msvc2019-win64.zip

使用管理员权限打开exe进行安装 

add PCL to the system Path for all users
弹出对话框是否安装这个设备软件PrimeSense 选择不安装

解压pcl-1.12.1-pdb-msvc2019-win64.zip 

解压得到的pdb文件都放入安装目录的bin目录下  
C:\Program Files\PCL 1.12.1\bin

重装OpenNI 

C:\Program Files\PCL 1.12.1\3rdParty\OpenNI2\OpenNI-Windows-x64-2.2.msi  
先执行卸载，然后双击它重新安装  
指定安装目录C:\Program Files\PCL 1.12.1\3rdParty\OpenNI2

配置环境第二步：vs2019新建项目配置pcl

正常新建c++控制台应用 

项目-右键-属性进入设置 
选择配置：Debug还是Release，活动平台选择X64  

分别在Debug-X64和Release-X64中修改包含目录、库目录、链接器中的附加依赖项  

debug和release的附加依赖项有区别，另外两项包含目录和库目录是一样的  

1. 通用属性 -> VC++目录 修改 常规->包含目录 和 库目录：  
2. 通用属性 -> 链接器 -> 输入 修改 附加依赖项 
3. 通用属性 -> C/C++ -> 预处理器 编辑预处理器定义  
增加  
BOOST_USE_WINDOWS_H 
NOMINMAX 
_CRT_SECURE_NO_DEPRECATE
_CRT_SECURE_NO_WARNINGS 

4. 通用属性 -> C/C++ -> 所有选项 编辑SDL检查 为 否(/sdl-)

5. 复制测试代码  

6. 点击F5Debug，如果提示找不到OpenNI2.dll,就去C:\Program Files\PCL 1.12.1\3rdParty\OpenNI2\Tools中把OpenNI2.dll放置到项目的debug文件夹中。

    cloud.height   = 1;  
    cloud.is_dense = false;  
    cloud.points.resize (cloud.width * cloud.height);  
    for (std::size_t i = 0; i < cloud.points.size (); ++i) 
    {    
        cloud.points[i].x = 1024 * rand () / (RAND_MAX + 1.0f);    
        cloud.points[i].y = 1024 * rand () / (RAND_MAX + 1.0f);    
        cloud.points[i].z = 1024 * rand () / (RAND_MAX + 1.0f);
    }  
        pcl::io::savePCDFileASCII ("test_pcd.pcd", cloud);  
        std::cerr << "Saved " << cloud.points.size () << " data points to test_pcd.pcd." << std::endl;  
        for (std::size_t i = 0; i < cloud.points.size (); ++i)    
             std::cerr << "    " << cloud.points[i].x << " " << cloud.points[i].y << " " << cloud.points[i].z << std::endl;  
        return (0);
}

    char strfilepath[256] = "rabbit.pcd";
    //加载点云文件
    if (-1 == pcl::io::loadPCDFile(strfilepath, *cloud)) {
        cout << "error input!" << endl;
        return -1;
    }

    // 控制台输出点云文件内容 
    // 滤波，剪裁，拟合等其他操作是不是调用类似的算法进行操作？
    std::cout << "Loaded: " << cloud->width * cloud->height << " data points from rabbit.pcd with the following fields:" << std::endl;
    // cout << cloud->points.size() << endl;
    for (size_t i = 0; i < cloud->points.size(); ++i) {
        std::cout << "      " << cloud->points[i].x << "   " << cloud->points[i].y << "   " << cloud->points[i].z << "   " << std::endl;
    }

    //创建viewer对象 可视化显示点云
    pcl::visualization::CloudViewer viewer("Cloud Viewer");   
    viewer.showCloud(cloud);
    viewer.runOnVisualizationThreadOnce(viewerOneOff);
    system("pause");
    return 0;
}

滚下滚轮，一个兔子出来了 

PointCloud是PCL中的一个基类，pcl::PointCloud<pcl::PointXYZ>::Ptr是一个Boost共享指针  

关于PointCloud中的数据域：  

width（int），如果是无组织，无结构的点云数据，表示点云的个数；如果是有结构的点云数据，表示点云数据集一行的点数。   
height（int），如果是无结构的点云数据，height=1；如果是有结构的点云数据，height表示点云总行数。  
points（std::vector）存储了数据类型为PointT的一个动态数组。  
PointXYZ 是最常见的一个点数据类型，它只包含三维X,Y,Z坐标信息　X：points[i].x  
size_t 整型，保存一个整数，记录一个大小（size）  
points.size（） 表示点云数据大小

PCD文件由文件头和数据域组成，文件头存储了点云数据的字段格式信息
读入数据后，可以遍历点云数据，并访问各个域上的值。
