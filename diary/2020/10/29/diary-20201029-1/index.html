<template>
    <el-input type="text" :value="value" @input="handleInput" > </el-input>
</template>
<script>

export default {
  name: 'newInput',
  props: {
    value: ''
  },
  data () {
    return {

    }
  },
  created: function () {
    console.log('newInput created')
  },
  methods: {
    handleInput (val) {
      this.$emit('input', val)
    }
  }
}
</script>

万能馅料搭配公式

当你吃腻了xx馅，想要换个新口味但是又不知道怎么搭配蔬菜，照本文公式来。让你的馅料够鲜！！
万能馅料搭配公式即：

好吃馅料=打底菜(肉或蛋)+口感菜1+口感菜2+特殊香气的菜

大概的解释一下

  • 打底菜:指肉。
  • 口感菜1:香气弱、口感软的菜。其中大量汁水蔬菜调馅前需要挤出水分。
  • 口感菜2:口感有点脆脆的，香气比1稍微重一点。
  • 特殊香气的菜:顾名思义香气非常浓重，不喜欢吃的人，看名字都能晕过去。
常见打底菜：猪肉、羊肉、牛肉、鱼肉、

常见口感菜1：比如冬瓜，西葫芦、南瓜、丝瓜、茄子、紫甘蓝、白菜、包菜、粉条、豆制品（老豆腐、千张、油豆腐）等

常见口感菜2：比如莲藕，荸荠，白萝卜、胡萝卜、西蓝花、黄瓜、木耳、玉米等

常见特殊香气的菜：比如茴香、芹菜、香菜、荠菜、芹菜、菠菜、番茄、韭菜、香菇等。

接下来举几个例子看看如何实际应用。

茴香馅

基础款肉+特殊香气的菜，这里蔬菜是主角有独特香气，肉是辅助负责口感。 

1.茴香去根，用加盐的清水浸泡10分钟，洗净、沥干水分、切末、备用。 
2.肉馅分3次加葱姜水，搅拌到充分吸收，再加盐、酱油、盐、味精、料酒、白糖，顺一个方向搅打上劲。 
3.最后加入香油打匀后，加入茴香即可。

韭菜馅

基础款肉+特殊香气的菜。

1.肉馅加盐、鸡精、五香粉、生抽、老抽、白胡椒粉、蚝油、香油、一个鸡蛋。少量多次的加入葱姜水，顺一个方向搅打上劲，放冰箱冷藏。 
2.韭菜清洗干净，切小粒，加入盐均匀搅拌，使韭菜表面出水，再将水分倒掉。(避免后期水越出越多，肉馅包不起来) 
3.韭菜和肉馅混合均匀即可。

萝卜猪肉馅

肉香主导，口感菜辅助，使肉馅清爽不腻歪。

1.锅中坐水烧开，将萝卜擦成丝，稍加焯水后捞出挤干水分。(焯水去除萝卜异味，冷水浸泡可保持萝卜脆爽口感) 
2.肉馅加盐、生抽、蚝油、糖、一个鸡蛋搅拌，少量多次加入葱姜水，顺一个方向搅打上劲，直到水完全吸收。
3.最后放入大葱碎，萝卜丝，淋上香油拌匀即可。

猪肉白菜馅

1.白菜洗净，切丝，再切成粒，加适量盐抓揉，腌制10分钟左右，让白菜充分出水。 
2.取一块蒸笼用的布，将白菜粒放上去，把布的四角收起来，攥紧，将白菜里的水分攥出来为止。 
3.猪肉馅依次放入盐、鸡精、五香粉、生抽 、老抽、蚝油、一个鸡蛋抓拌均匀，然后少量多次的加入葱姜水，顺时针搅拌上劲，最后将白菜馅倒入，再加香油拌匀即可。

芹菜木耳猪肉馅

芹菜是主要香气来源，木耳也是辅助，增加脆爽的口感。

1 黑木耳提前泡发，洗净后切碎。芹菜叶摘下来洗净。 
2 锅里水开后下芹菜焯水，芹菜一软就可以捞出来，晾凉后切碎，用手把水分挤出来，不用挤得太干。 
3 肉馅，加盐、鸡精、五香粉、生抽 、老抽、白胡椒粉、、蚝油、香油 ，一个鸡蛋，顺时针搅拌上劲后，再加木耳、芹菜混合搅拌匀就可以包了。

牛肉胡萝卜洋葱馅

1 胡萝卜用擦子擦成丝，用刀剁碎，洋葱切碎。 
2 起锅热油，分别吧胡萝卜碎和香菇碎炒香，稍微的放一点点盐入个底味儿，一点点就够了。 
3 肉馅加一个鸡蛋、盐、鸡精、五香粉、生抽 、老抽、白胡椒粉、蚝油、 少量多次的加入葱姜水，顺时针搅拌上劲后加香油封面。 
4 加炒好胡萝卜 洋葱，搅拌均匀即可开包。

香菇玉米猪肉馅

1、肉馅加适量的盐、胡椒粉、生抽、料酒、鸡粉、蚝油、一个鸡蛋抓拌均匀，少量多次的加入葱姜水，顺时针搅拌上劲，最后加香油封面。 
2、香菇洗净后切成末，玉米剥粒，也可以用超市冷冻的玉米粒。 
3、将香菇、玉米放入腌制好的肉馅中，继续朝一个方向拌匀即可。

莲藕香菇肉馅

1 莲藕削皮，切成小碎丁，淘洗一下沥干水分。 
2 干香菇泡发后切成小碎丁，起锅烧油炒至干香。 
3 把炒好的香菇丁和肉馅和莲藕丁混合，放盐、鸡精、五香粉、生抽 、老抽、白胡椒粉、、蚝油、香油， 顺时针搅拌均匀至上劲。

韭菜鸡蛋粉条馅

1 韭菜洗净切末，加玉米油拌匀。粉条用温水泡软切碎。 
2 把所有的盐都加入到鸡蛋液里，后面尽可能的不再放盐了，因为韭菜中加盐最容易出汤。鸡蛋液边倒边搅，炒成鸡蛋碎。 
3  韭菜末、粉条末、鸡蛋末放在一起，加蚝油、糖、鸡精、香油搅拌均匀就可以包了

黄瓜鸡蛋木耳馅

1、木耳泡发切丝。 
2、黄瓜擦丝后用纱布挤出水，放油拌匀。 
3、鸡蛋加盐炒散越碎越好 
4、配上黄瓜丝，木耳丝，加鸡精，五香粉，芝麻油拌匀。

西葫芦鸡蛋萝卜馅

1 西葫芦擦丝，胡萝卜切末，放少许盐稍稍杀软，攥出水分后拌油。 
2 鸡蛋加盐炒散炒碎。 
3 鸡蛋碎和西葫芦拌匀，加鸡精、蚝油、香油拌匀。

包菜香菇豆腐馅

1 包菜切碎攥出水分后拌油。 
2.老豆腐两面煎至微黄，切碎。 
3 干香菇泡发后切碎，放油炒出香味。 
4.把包菜、豆腐、香菇依次加入搅拌，放适量的盐、鸡精、蚝油、香油搅拌即可。

香菇鸡蛋胡萝卜木耳粉条馅

1 香菇、木耳、粉条泡发后切碎。胡萝卜切丁。 
2 锅里倒油，放花椒爆出香味捞出来，倒入胡萝卜，香菇、木耳、粉条翻炒，最后加入鸡蛋碎，放盐、鸡精、蚝油调味，盛出即可。

油豆腐荸荠香菇馅

1. 油豆腐丁，荸荠丁、香菇丁下锅炒出香味，加盐、生抽、糖、蚝油、鸡精、香翻炒均匀即可。

胡萝卜香菇豆腐干木耳玉米粒馅

胡萝卜丁、香菇丁、豆腐干丁、木耳末，玉米粒在锅里翻炒出香味，加盐、生抽、糖、蚝油、鸡精、香翻炒均匀即可。

肉馅小技巧

  1. 葱姜水的做法：葱姜切碎后用凉白开浸泡半小时左右，可以多抓揉几下，把葱姜汁捏出来，最后过滤出来的水就是葱姜水。
  2. 肉的肥瘦比例都是3:7或者4:6最好吃。菜和肉的比例是1:1最好吃。
  3. 肉馅想要饱满多汁是一定要打葱姜水的。(有些绝不吃姜的嘴巴再细的姜丝也能吃出来，所以用葱姜水去腥提鲜。)。
  4. 少量多次的加入肉馅，朝同一方向分次搅拌到肉馅黏连。水量加多少根据肉的状态来。
  5. 判断肉馅水打的够不够的方法：把肉馅堆到5cm的高度，斜插进去个筷子，筷子慢慢落下去就是刚好，迟迟落不下去就太干，继续打水。筷子吧唧一下倒了是太稀。如果没有肉或者菜可以加进去，就放在冰箱冷藏一下，这样才好包。
  6. 肉馅要等到包的时候再把素菜和肉混合，最后放香油。
  7. 加了生鸡蛋的饺子馅吃起来会更滑嫩。

素馅小技巧

  1. 肉馅要后放油，素馅则恰恰相反要先放油拌菜，锁住水分。
  2. 不要花生油橄榄油，要玉米油色拉油这类没味儿的。最少要馅料的10%，这样馅料吃起来才香。
  3. 素馅不要放老抽。
  4. 最后放盐，如果有加鸡蛋碎，就把盐都放在鸡蛋里，一直到馅包完，菜都不会出水。
  5. 虾仁是个比较中立的存在，加在素馅和肉馅里面都特别好吃。最好不要切碎，在饺子里吃到整个虾仁才满足。虾皮和海米也都可以！
