Python图像处理之4_Pillow
Python Pillow
发布日期:   2020-09-16
作者:   sean
文章字数:   1.2k
阅读时长:   5 分

Pillow是Python中的一个图形处理库。

获取图像信息

from PIL import Image 
im = Image.open('12.jpg')
im.save('red.png') 
print('图像的格式：', im.format)
print('图像的大小：', im.size)
print('图像的宽度：', im.width)
print('图像的高度：', im.height) 
print('获取某个像素点的颜色值：', im.getpixel((100, 100))) 
im.show()

创建一个简单的图像

from PIL import Image 
im=Image.new('RGB',(150,150),'red') 
im.save('red.png')

blend图像混合

透明度混合主要是使用Image中的blend(im1, im2, alpha)方法，
im1：Image对象，在混合的过程中，透明度设置为（1-apha）
im2：Image对象，在混合的过程中，透明度设置为（apha）
alpha：透明度，取值是0-1。当透明度为0是，显示im1对象；当透明度为1时，显示im2对象
注意：im1和im2的大小必须一样，且mode都为RGB

from PIL import Image
# 打开im1
im1 = Image.open('21.jpg').convert(mode='RGB')
# 创建一个和im1大小一样的图像
im2 = Image.new('RGB', im1.size, 'red')
# 混合图片，并显示
Image.blend(im1, im2, 0.2).show()

遮罩图像混合

我们可以通过Image.composite(im1, im2, mask) 方法实现遮罩混合。三个参数都是Image对象，该方法的作用就是使用mask来混合im1和im2。
注意：im1、im2和mask的大小必须一样 

from PIL import Image

im1 = Image.open('1.jpg')                # 打开图像1 
im2 = Image.open('2.jpg')                # 打开图像2 
im2.resize(im1.size)                    # 重新设置im2的大小 
r, g, b = im2.split()                    # 将图像2的三个色道分离，其中r、g、b都为Image对象 
Image.composite(im1, im2, b).show()     # 遮罩混合

图像缩放

按像素缩放通过Image.eval(im1,fun)方法实现，其中im1为我们老生常谈的Image对象了；第二个为一个方法（函数），该函数传入一个参数，即像素点。该函数会对图片中每个像素点进行函数内的操作 

from PIL import Image

im = Image.open('21.jpg')                # 打开一张图像
Image.eval(im, lambda x:x*2).show()        # 对该图像每个像素点进行*2处理

按尺寸缩放是通过Image对象的thumbnail()方法实现 

from PIL import Image 
im1 = Image.open('1.jpg') 
im2 = im1.copy() 
im2.thumbnail((50, 50))
im2.save('result.png')
print("im1的大小", im1.size)
print('im2的大小', im2.size)

复制、剪切、粘贴

粘贴的实现主要是通过Image对象的paste(im,box,mask)方法，其中im为Image对象；box为要粘贴到的区域；mask为遮罩（我也不知道啥是遮罩）。其中box的参数有三种形式：
(x1, y1)：将im左上角对齐(x1,y1)点，其余部分粘贴，超出部分抛弃
(x1, x2, y1, y2)：将im粘贴至此区域
None：此时im必须与源图像大小一致

from PIL import Image
im = Image.open('1.jpg')
im1 = im.copy()                                    # 复制两份
im2 = im.copy()
im_crop = im1.crop((200, 200, 300, 300))        # 剪切图片
im2.paste(im_crop, (30, 30))                    # 粘贴图片
im2.show()
im2.rotate(90).show()                            # 旋转图片

图像的分离和合并

Image对象的split()方法，将图像的RGB三个通道分离，并返回三个Image对象 

from PIL import Image
im = Image.open('1.jpg')
r, g, b = im.split()        # 分离通道，返回3个Image对象
r.show()
g.show()
b.show()

合并是通过Image.merge(mode, bands)方法实现的，其中mode为模式，bands为通道列表，传入一个列表类型数据。 

from PIL import Image

im1 = Image.open('1.jpg')
im2 = Image.open('2.jpg')
im2.resize(im1.size)             # 让im2大小和im1一样
r1, g1, b1 = im1.split()        # 将两个图像分别分离
r2, g2, b2 = im2.split() 
im3 = Image.merge('RGB', [r1, g2, b1])    # 合并图像
im3.show()

几个常用图像滤镜

高斯模糊也叫高斯平滑，是通常用它来减少图像噪声以及降低细节层次。这种模糊技术生成的图像，其视觉效果就像是经过一个毛玻璃在观察图像，这与镜头焦外成像效果散景以及普通照明阴影中的效果都明显不同

from PIL import Image, ImageFilter
im1 = Image.open('12.jpg')                                    # 打开图像
img = Image.new('RGB', (im1.width*2, im1.height), 'red')    # 创建一个im1两倍宽的图像

im2 = im1.filter(ImageFilter.GaussianBlur)                    # 高斯模糊滤镜
#im2 = im1.filter(ImageFilter.CONTOUR)                        # 轮廓滤镜
#im2 = im1.filter(ImageFilter.DETAIL)                        # 细节滤镜
#im2 = im1.filter(ImageFilter.EDGE_ENHANCE)                    # 边缘增强滤镜
#im2 = im1.filter(ImageFilter.EDGE_ENHANCE_MORE)            # 边缘增强plus滤镜
#im2 = im1.filter(ImageFilter.EMBOSS)                        # 浮雕滤镜
#im2 = im1.filter(ImageFilter.FIND_EDGES)                    # 寻找边缘滤镜 
#im2 = im1.filter(ImageFilter.SMOOTH)                        # 平滑滤镜

img.paste(im1, (0, 0))                                        # 将im1粘贴到img上
img.paste(im2, (im1.width, 0))                                # 将im2（高斯模糊后的图像）粘贴到img上
img.show()
